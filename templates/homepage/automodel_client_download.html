Instalation Instruction (Ubuntu Linux 16.04):


Order to use the AutoModel, the following components are required:

  • Python (Already included in Ubuntu);
  • RPyC Library (Library for server communication);
  • WxPython Library (Graphics rendering library);
  • VMD.

RPyC Installation:

Open the terminal and run the following commands:

				sudo apt-get install python-pip
				sudo easy_install rpyc==3.3.0

WxPython Installation:

				sudo add-apt-repository ppa:nilarimogard/webupd8
				sudo apt-get update
				sudo apt-get install python-wxgtk2.8 wx2.8-i18n

VMD Installation:

Download the VMD here, unpack it and enter in his folder using the terminal. Enter the following commands:

				./configure
				cd src
				sudo make install

After installing these components, just unzip the automodel-client (preferably in home folder) and run the file automodel.sh (2x clicking on or by the terminal with the command sh automodel.sh).
